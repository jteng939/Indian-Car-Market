# Indian-Car-Market
This project analyzes pricing drivers in the Indian automobile market using statistical modeling. My partner and I applied distribution analysis, chi-square tests, and nonlinear regression to identify how vehicle attributes influence selling price with a focus on business implications.

My contribution: Conducted distribution and regression analysis using JMP, supported exploratory analysis in Excel, and contributed to the project write-up and interpretation of results.


Data Source: [Kaggle](https://www.kaggle.com/datasets/sukhmandeepsinghbrar/car-price-prediction-dataset/data 
)
